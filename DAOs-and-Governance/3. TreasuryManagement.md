# Treasury Management

## Introduction

Treasury management is one of the most critical functions of a DAO. The treasury holds the organization's assets and funds operations, development, grants, and strategic initiatives. Effective treasury management requires balancing security, diversification, liquidity, and growth while maintaining transparency and accountability.

## Treasury Architecture

### Multi-Signature Wallet (Gnosis Safe)

The most common and secure treasury implementation uses multi-signature wallets.

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

contract MultiSigTreasury {
    address[] public owners;
    uint256 public required;
    
    struct Transaction {
        address to;
        uint256 value;
        bytes data;
        bool executed;
        uint256 confirmations;
    }
    
    mapping(uint256 => Transaction) public transactions;
    mapping(uint256 => mapping(address => bool)) public confirmations;
    uint256 public transactionCount;
    
    event Submission(uint256 indexed transactionId);
    event Confirmation(address indexed sender, uint256 indexed transactionId);
    event Execution(uint256 indexed transactionId);
    event ExecutionFailure(uint256 indexed transactionId);
    
    modifier onlyOwner() {
        require(isOwner(msg.sender), "Not owner");
        _;
    }
    
    modifier transactionExists(uint256 transactionId) {
        require(transactions[transactionId].to != address(0), "Transaction does not exist");
        _;
    }
    
    modifier notExecuted(uint256 transactionId) {
        require(!transactions[transactionId].executed, "Transaction already executed");
        _;
    }
    
    modifier notConfirmed(uint256 transactionId) {
        require(!confirmations[transactionId][msg.sender], "Transaction already confirmed");
        _;
    }
    
    constructor(address[] memory _owners, uint256 _required) {
        require(_owners.length > 0, "Owners required");
        require(_required > 0 && _required <= _owners.length, "Invalid required number");
        
        owners = _owners;
        required = _required;
    }
    
    function submitTransaction(address to, uint256 value, bytes memory data)
        public
        onlyOwner
        returns (uint256)
    {
        uint256 transactionId = transactionCount;
        transactions[transactionId] = Transaction({
            to: to,
            value: value,
            data: data,
            executed: false,
            confirmations: 0
        });
        transactionCount++;
        
        emit Submission(transactionId);
        confirmTransaction(transactionId);
        
        return transactionId;
    }
    
    function confirmTransaction(uint256 transactionId)
        public
        onlyOwner
        transactionExists(transactionId)
        notConfirmed(transactionId)
    {
        confirmations[transactionId][msg.sender] = true;
        transactions[transactionId].confirmations++;
        
        emit Confirmation(msg.sender, transactionId);
        
        executeTransaction(transactionId);
    }
    
    function executeTransaction(uint256 transactionId)
        public
        onlyOwner
        transactionExists(transactionId)
        notExecuted(transactionId)
    {
        Transaction storage txn = transactions[transactionId];
        
        if (txn.confirmations >= required) {
            txn.executed = true;
            (bool success, ) = txn.to.call{value: txn.value}(txn.data);
            
            if (success) {
                emit Execution(transactionId);
            } else {
                emit ExecutionFailure(transactionId);
                txn.executed = false;
            }
        }
    }
    
    function isOwner(address account) public view returns (bool) {
        for (uint256 i = 0; i < owners.length; i++) {
            if (owners[i] == account) {
                return true;
            }
        }
        return false;
    }
    
    receive() external payable {}
}
```

### Governance-Controlled Treasury

Treasury controlled by governance proposals.

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

import "@openzeppelin/contracts/token/ERC20/IERC20.sol";
import "@openzeppelin/contracts/access/Ownable.sol";

contract GovernanceTreasury is Ownable {
    event FundsTransferred(address indexed token, address indexed to, uint256 amount);
    event FundsReceived(address indexed from, uint256 amount);
    
    // Only governance (owner) can execute transfers
    function transfer(
        address token,
        address to,
        uint256 amount
    ) external onlyOwner {
        if (token == address(0)) {
            // ETH transfer
            payable(to).transfer(amount);
        } else {
            // ERC20 transfer
            IERC20(token).transfer(to, amount);
        }
        
        emit FundsTransferred(token, to, amount);
    }
    
    function batchTransfer(
        address[] memory tokens,
        address[] memory recipients,
        uint256[] memory amounts
    ) external onlyOwner {
        require(
            tokens.length == recipients.length && recipients.length == amounts.length,
            "Array length mismatch"
        );
        
        for (uint256 i = 0; i < tokens.length; i++) {
            if (tokens[i] == address(0)) {
                payable(recipients[i]).transfer(amounts[i]);
            } else {
                IERC20(tokens[i]).transfer(recipients[i], amounts[i]);
            }
            
            emit FundsTransferred(tokens[i], recipients[i], amounts[i]);
        }
    }
    
    function execute(
        address target,
        uint256 value,
        bytes calldata data
    ) external onlyOwner returns (bytes memory) {
        (bool success, bytes memory result) = target.call{value: value}(data);
        require(success, "Execution failed");
        return result;
    }
    
    function getBalance(address token) external view returns (uint256) {
        if (token == address(0)) {
            return address(this).balance;
        } else {
            return IERC20(token).balanceOf(address(this));
        }
    }
    
    receive() external payable {
        emit FundsReceived(msg.sender, msg.value);
    }
}
```

## Treasury Diversification

### Asset Allocation Strategy

```python
class TreasuryAllocation:
    def __init__(self):
        self.allocation = {
            'stablecoins': 0.40,      # 40% - Operational runway
            'native_token': 0.30,      # 30% - Governance token
            'blue_chip_crypto': 0.20,  # 20% - ETH, BTC
            'defi_positions': 0.10     # 10% - Yield generation
        }
    
    def calculate_target_allocation(self, total_value):
        """Calculate target amounts for each asset class"""
        return {
            asset: total_value * percentage
            for asset, percentage in self.allocation.items()
        }
    
    def rebalance_needed(self, current_allocation, target_allocation, threshold=0.05):
        """Check if rebalancing is needed (>5% deviation)"""
        for asset in current_allocation:
            current = current_allocation[asset]
            target = target_allocation[asset]
            
            if abs(current - target) / target > threshold:
                return True
        
        return False
```

```solidity
        require(msg.sender == governance, "Only governance");
        _;
    }

            token: token,
            targetPercentage: targetPercentage,
            currentAmount: 0
        });
```

## Budget Management

### Streaming Payments

Continuous payment streams for contributors.

```solidity
// SPDX-License-Identifier: MIT
pragma solidity ^0.8.0;

contract StreamingPayments {
    struct Stream {
        address recipient;
        uint256 amountPerSecond;
        uint256 startTime;
        uint256 endTime;
        uint256 claimed;
    }
    
    mapping(uint256 => Stream) public streams;
    uint256 public streamCount;
    
    IERC20 public token;
    address public treasury;
    
    event StreamCreated(uint256 indexed streamId, address indexed recipient, uint256 amountPerSecond);
    event StreamClaimed(uint256 indexed streamId, uint256 amount);
    
    modifier onlyTreasury() {
        require(msg.sender == treasury, "Only treasury");
        _;
    }
    
    function createStream(
        address recipient,
        uint256 totalAmount,
        uint256 duration
    ) external onlyTreasury returns (uint256) {
        uint256 streamId = streamCount++;
        uint256 amountPerSecond = totalAmount / duration;
        
        streams[streamId] = Stream({
            recipient: recipient,
            amountPerSecond: amountPerSecond,
            startTime: block.timestamp,
            endTime: block.timestamp + duration,
            claimed: 0
        });
        
        emit StreamCreated(streamId, recipient, amountPerSecond);
        return streamId;
    }
    
    function claimStream(uint256 streamId) external {
        Stream storage stream = streams[streamId];
        require(msg.sender == stream.recipient, "Not recipient");
        
        uint256 claimable = getClaimableAmount(streamId);
        require(claimable > 0, "Nothing to claim");
        
        stream.claimed += claimable;
        token.transfer(stream.recipient, claimable);
        
        emit StreamClaimed(streamId, claimable);
    }
    
    function getClaimableAmount(uint256 streamId) public view returns (uint256) {
        Stream memory stream = streams[streamId];
        
        uint256 endTime = block.timestamp > stream.endTime
            ? stream.endTime
            : block.timestamp;
        
        uint256 elapsed = endTime - stream.startTime;
        uint256 totalEarned = elapsed * stream.amountPerSecond;
        
        return totalEarned - stream.claimed;
    }
}
```

### Milestone-Based Grants

Release funds based on milestone completion.

```solidity
contract MilestoneGrants {
    struct Grant {
        address recipient;
        uint256 totalAmount;
        uint256 released;
        Milestone[] milestones;
    }
    
    struct Milestone {
        string description;
        uint256 amount;
        bool completed;
        bool approved;
    }
    
    mapping(uint256 => Grant) public grants;
    uint256 public grantCount;
    
    address public governance;
    
    event GrantCreated(uint256 indexed grantId, address indexed recipient, uint256 totalAmount);
    event MilestoneCompleted(uint256 indexed grantId, uint256 milestoneIndex);
    event MilestoneApproved(uint256 indexed grantId, uint256 milestoneIndex);
    event FundsReleased(uint256 indexed grantId, uint256 amount);
    
    function createGrant(
        address recipient,
        uint256[] memory milestoneAmounts,
        string[] memory milestoneDescriptions
    ) external returns (uint256) {
        require(msg.sender == governance, "Only governance");
        
        uint256 grantId = grantCount++;
        Grant storage grant = grants[grantId];
        grant.recipient = recipient;
        
        uint256 totalAmount = 0;
        for (uint256 i = 0; i < milestoneAmounts.length; i++) {
            grant.milestones.push(Milestone({
                description: milestoneDescriptions[i],
                amount: milestoneAmounts[i],
                completed: false,
                approved: false
            }));
            totalAmount += milestoneAmounts[i];
        }
        
        grant.totalAmount = totalAmount;
        
        emit GrantCreated(grantId, recipient, totalAmount);
        return grantId;
    }
    
    function submitMilestone(uint256 grantId, uint256 milestoneIndex) external {
        Grant storage grant = grants[grantId];
        require(msg.sender == grant.recipient, "Not recipient");
        require(!grant.milestones[milestoneIndex].completed, "Already completed");
        
        grant.milestones[milestoneIndex].completed = true;
        emit MilestoneCompleted(grantId, milestoneIndex);
    }
    
    function approveMilestone(uint256 grantId, uint256 milestoneIndex) external {
        require(msg.sender == governance, "Only governance");
        
        Grant storage grant = grants[grantId];
        Milestone storage milestone = grant.milestones[milestoneIndex];
        
        require(milestone.completed, "Not completed");
        require(!milestone.approved, "Already approved");
        
        milestone.approved = true;
        grant.released += milestone.amount;
        
        // Transfer funds
        IERC20(token).transfer(grant.recipient, milestone.amount);
        
        emit MilestoneApproved(grantId, milestoneIndex);
        emit FundsReleased(grantId, milestone.amount);
    }
    
    IERC20 public token;
}
```

```python
        runway_months = total_usd / monthly_burn_rate
```

```solidity
            coverage: coverage,
            premium: premium,
            startTime: block.timestamp,
            duration: duration,
            active: true
        });

    }
    
    function fileClaim(uint256 policyId, uint256 amount) external {
        require(msg.sender == treasury, "Only treasury");
        
        Policy storage policy = policies[policyId];
        require(policy.active, "Policy not active");
        require(amount <= policy.coverage, "Exceeds coverage");

    }
```

### Treasury Security Checklist

```markdown
## Security Measures

1. Multi-Signature Requirements
   - [ ] Minimum 3-of-5 signers for large transactions
   - [ ] Geographic distribution of signers
   - [ ] Hardware wallet usage required
   - [ ] Regular signer rotation

2. Transaction Limits
   - [ ] Daily withdrawal limits
   - [ ] Per-transaction limits
   - [ ] Timelock for large transfers
   - [ ] Whitelist for recipient addresses

3. Monitoring and Alerts
   - [ ] Real-time transaction monitoring
   - [ ] Anomaly detection
   - [ ] Balance alerts
   - [ ] Regular audits

4. Diversification
   - [ ] No more than 40% in any single asset
   - [ ] Stablecoin reserves for operations
   - [ ] Risk-adjusted yield strategies
   - [ ] Regular rebalancing

5. Documentation
   - [ ] All transactions documented
   - [ ] Budget tracking
   - [ ] Monthly reports
   - [ ] Annual audits
```

## Conclusion

Effective treasury management is critical for DAO sustainability and success. Key principles include:

- Security through multi-signature controls and timelocks
- Diversification across asset classes
- Sustainable yield generation
- Transparent reporting and accountability
- Risk management and insurance
- Efficient budget allocation

**Treasury Management Framework:**
1. Establish clear policies and procedures
2. Implement secure custody solutions
3. Diversify assets appropriately
4. Generate sustainable yield
5. Monitor and report regularly
6. Maintain adequate runway
7. Plan for contingencies
